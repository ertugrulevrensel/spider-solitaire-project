# Spider Solitaire Project

Live: https://evrensel-spider-solitaire-project.netlify.app/

## Used Technologies

- React
- React Beautiful DND
- Contex API
- Eslint
- Babel
- React-Timer-Hook
- SASS

## Clone Project

```console
git clone https://github.com/ertugrulevrensel/spider-solitaire-project.git
```

## Run Project

In the project directory you can run:

```console
npm install
npm start
```
